Aria2 文件夹放置在/system/xbin目录下，别忘了权限需要设置。
终端模拟器打开后执行命令 "/system/xbin/Aria2/bin/aria2" 即可打开控制页面。
或者在xbin目录下做个链接也可以。


The main file list Aria2 should be placed in /system/xbin/
And don't foget set permission of executive binary.

Use command /system/xbin/Aria2/bin/aria2 in terminal emulator to get in controller.
Of couse you can make a link in xbin by command "ln -s /system/xbin/Aria2/bin/aria2 /system/xbin/".
So next time command aria2 would work too.
